# 🌐 Nandhakumar M — Personal Portfolio

A modern, responsive **personal portfolio website** for **Nandhakumar M**, a
Software Developer (Java & Spring Boot) at Cloudstier Technologies. Built with
plain **HTML**, **CSS**, and **JavaScript** — no build step required.

## 🚀 Features

- 🦸 **Hero section** with gradient typography, profile blob, and quick CTAs
- 📊 **Animated statistics** (Years of Learning, Projects, Technologies)
- 👤 **About** section with role / company / education highlights
- 🧩 **Skill cards** grouped by category (Languages, Backend, Frontend,
  Databases, Tools, Core Competencies)
- 🗓 **Experience timeline** for work history and education
- 📂 **Project cards** with tech tags and hover effects
- 🌙 **Dark / light theme** toggle (remembers your choice)
- 🔝 Scroll-to-top button, scroll-spy nav, and smooth scroll-reveal animations
- 📱 Fully **responsive** and mobile-friendly
- 🔍 **SEO-ready**: meta tags, Open Graph, and JSON-LD structured data
- 📞 Contact form powered by [Formspree](https://formspree.io)
